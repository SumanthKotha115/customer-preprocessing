# Customer Data Preprocessing

## Tasks Completed
- Data Cleaning (missing values, duplicates)
- Encoding categorical data
- Feature scaling

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn

## Steps Performed
1. Filled missing age using median
2. Filled missing city using mode
3. Removed duplicate rows
4. One-hot encoded city
5. Label encoded gender
6. Applied Min-Max Scaling and Standardisation
